Vous avez une erreur du type "python/pip n'est pas reconnu en tant que commande interne"?


C'est simple, vous n'avez pas installé Python correctement (ou ne l'avez pas installé du tout)!
Rendez-vous sur le site officiel de Python puis téléchargez-le: https://www.python.org/downloads/release/python-390/
Ensuite, éxécutez l'installateur.

A partir de ce moment, vous aurez deux cas:

1: Vous avez déjà cette version de Python, et trois choix sont visibles: "Modify", "Repair" et "Uninstall".
Cliquez sur Modify. Une page s'affichera contenant plusieurs paramètres (dont "pip")!
Cochez-les tous et appuyez sur "Next". Une autre page s'ouvrira avec plusieurs paramètres. Cochez "Add Python to the environment variables" si ce n'est pas déjà fait.
Ensuite, vous pouvez appuyer sur "Install".


2: Vous n'avez pas cette version de Python, et il y'à seulement "Install now" et "Customize installation" qui sont affichés.
Tout d'abord, cochez "add Python 3.9 to path" tout en bas.
Ensuite, appuyez sur "Customize installation". Une page s'affichera avec plusieurs paramètres (dont "pip")!
Cochez-les tous et appuyez sur "Next", puis ne touchez à rien et appuyez sur "Install".


Voilà! Vos problèmes sont maintenant résolus.
