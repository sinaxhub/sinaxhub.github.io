Votre fichier Python ne se lance pas?


Avant de commencer, décompressez le fichier ".zip" que vous avez téléchargé sur mon GitHub si ce n'est pas déjà fait, puis réessayez d'éxécuter le programme.

Ce problème peut être causé par plusieurs raisons. La plus fréquente est l'absence des modules nécessaires au fonctionnement du programme.
Pour y remédier, vous allez exécuter votre fichier Python dans un terminal et installer les modules nécessaires.
Ouvrez l'explorateur de fichiers et rendez-vous dans le dossier ou se trouve votre programme.
En haut, vous verrez le chemin d'accès (la path) du dossier, ex: C:/Users/billy/Desktop/Vulture/vulture.py
Copiez cette path, puis cherchez "cmd" dans la recherche Windows et appuyez sur entrée.
Un terminal sera ouvert. Entrez "cd path_du_dossier" et appuyez sur entrée.
Ensuite, entrez "python nom_du_fichier" et appuyez sur entrée.
Votre erreur s'affichera.
Si l'erreur ressemble à ça: "ModuleNotFoundError: No module named 'nom_du_module'", alors entrez la commande suivante dans votre terminal:
"pip install nom_du_module"
Une barre de chargement s'affichera et le module sera installé!
Vous pourrez désormais éxécuter le fichier Python.
